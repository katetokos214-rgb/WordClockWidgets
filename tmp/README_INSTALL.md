# 📱 WordClockWidgets iOS App - Installation Guide

You've downloaded the compiled IPA file for WordClockWidgets iOS app!

## File Information
- **Name**: WordClockWidgets.ipa
- **Size**: ~10-15 MB
- **Target**: iPhone XS with iOS 14.0+
- **Status**: Ready to install

## Installation Methods

### Method 1: iMazing (Windows)

**This is the recommended method for Windows users!**

1. Download and install [iMazing](https://imazing.com/) on your Windows PC
2. Connect your iPhone to Windows via USB
3. Open iMazing
4. Click on "Install app" button
5. Select this IPA file
6. Wait for installation to complete
7. On iPhone: Settings > General > VPN & Device Management > Trust the developer

**Time**: Usually 2-5 minutes

### Method 2: Apple Configurator 2 (Mac)

1. Download [Apple Configurator 2](https://apps.apple.com/app/apple-configurator-2/id1037126344) from Mac App Store
2. Connect iPhone via USB
3. Drag the IPA file to device icon
4. Confirm installation

### Method 3: Xcode (Mac)

1. Download Xcode from Mac App Store
2. Connect iPhone via USB
3. Open Xcode
4. Menu: Window > Devices and Simulators
5. Select your iPhone
6. Drag the IPA file to device
7. Confirm installation

### Method 4: iTunes (Old macOS)

If you have iTunes on older Mac:
1. Connect iPhone
2. Open iTunes
3. Drag IPA file to iTunes window
4. Sync

## After Installation

1. **On your iPhone**, go to:
   - Settings
   - General
   - VPN & Device Management (or Device Management)
   - Find your Apple ID
   - Tap **Trust**

2. The app icon should now appear on your home screen
3. Tap to launch

## Troubleshooting

### "iPhone not detected"
- Try different USB cable
- Reconnect iPhone
- Trust the connection on iPhone screen
- Restart iPhone
- Restart your Windows PC

### "Installation failed"
- Check that you have Storage space on iPhone
- Try again with USB 2.0 port (not USB 3.0)
- Update iTunes/iMazing to latest version

### "App crashes after opening"
- Make sure you tapped Trust in Device Management
- Restart iPhone
- Reinstall the app

### "Trust button not appearing"
- Unlock iPhone
- Tap "Trust" on the popup dialog
- If no popup appears, reconnect the device
